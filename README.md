# robust-yolov8-brain-tumor-detection
Research project on an improved YOLOv8 framework for multi-class brain tumor detection with enhanced robustness and clinical interpretability. Code will be released after paper publication.

This repository contains the implementation of our research work on improving YOLOv8 for brain tumor detection using MRI images.

Status: Code will be released after paper publication.

## Overview
- Multi-class brain tumor detection
- Improved training stability
- Medical imaging focused evaluation

## Model
Based on YOLOv8 with custom modifications.
